CYFO INC Spear Phishing Analysis Report

1. Background

This project conducts a full forensic analysis of emails targeting the fictional company CYFO INC, identifying spear phishing activity and attacker attribution.

2. Data Processing Workflow

  • Extraction: Parsed .eml files using parser.py to extract structured fields including sender, receivers, IPs, content, etc.
  • Cleaning: Normalized email formats, parsed timestamps, cleaned MIME types.
  • Modeling & Analysis: Performed statistical modeling with pandas and sqlite3.
  • Visualization & Reporting: Generated chart visualizations and exported Markdown or HTML reports.

3. Spear Phishing Target Detection

By matching recipients against the employee list, 69 emails were identified as targeting CYFO staff.

4. Attacker Attribution

IP addresses extracted from headers were analyzed via WHOIS to determine origin organizations and countries.

5. Conclusions and Recommendations

  • Enhance security awareness training for high-risk departments like IT and HR
  • Implement SPF, DKIM, and DMARC policies on the email gateway
  • Blacklist attacker IPs at the firewall or SIEM level
  • Automate future forensic workflows for phishing analysis

6. Project Deliverables

  • Cleaned email data: cleaned_emails.json
  • Employee reference list: employees.csv
  • Visual charts: 3 charts including bar and pie plots
  • Final report: spear_report.md or exported PDF
